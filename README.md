# نرمافزار مدیریت پروژه‌های نرمافزاری

## 📋 معرفی پروژه

این نرمافزار یک سیستم جامع مدیریت پروژه‌های نرمافزاری است که با استفاده از Python و Tkinter طراحی و پیاده‌سازی شده است. این نرمافزار امکان مدیریت پروژه‌ها، گزارش‌گیری، احراز هویت کاربران و ویرایش متن را فراهم می‌کند.

## 🚀 ویژگی‌های کلیدی

- **سیستم احراز هویت امن**: ورود و ثبت‌نام کاربران با رمزنگاری SHA-256
- **مدیریت پروژه‌ها**: ثبت، ویرایش و حذف پروژه‌ها
- **مدیریت شرکت‌ها و کارفرمایان**: ثبت اطلاعات مشتریان
- **گزارش‌گیری پیشرفته**: انواع مختلف گزارش‌های مالی و زمانی
- **ویرایشگر متن**: ابزار ویرایش متن با قابلیت‌های پیشرفته
- **رابط کاربری فارسی**: طراحی شده برای کاربران فارسی‌زبان
- **امنیت بالا**: قفل حساب پس از تلاش‌های ناموفق

## 🔧 کلاس‌ها و توابع

### 1. کلاس اصلی برنامه (`main.py`)

#### کلاس `ProjectManagementApp`
کلاس اصلی که کل برنامه را مدیریت می‌کند.

**متدهای اصلی:**
- `__init__()`: مقداردهی اولیه برنامه
- `setup_ui()`: ایجاد رابط کاربری اصلی
- `create_menu_buttons()`: ایجاد دکمه‌های منو
- `show_login()`: نمایش پنجره ورود
- `show_register()`: نمایش پنجره ثبت‌نام
- `show_project_management()`: نمایش پنجره مدیریت پروژه
- `show_reports()`: نمایش پنجره گزارش‌گیری
- `run()`: اجرای برنامه

### 2. مدیریت احراز هویت (`auth_manager.py`)

#### کلاس `AuthManager`
مسئول مدیریت ورود، ثبت‌نام و امنیت حساب‌های کاربری.

**متدهای اصلی:**
- `load_users()`: بارگذاری کاربران از فایل
- `save_users()`: ذخیره کاربران در فایل
- `hash_password()`: رمزنگاری رمز عبور
- `is_account_locked()`: بررسی قفل بودن حساب
- `record_failed_login()`: ثبت تلاش ناموفق ورود
- `show_login_form()`: نمایش فرم ورود
- `show_register_form()`: نمایش فرم ثبت‌نام

**ویژگی‌های امنیتی:**
- رمزنگاری SHA-256 برای رمزهای عبور
- قفل حساب پس از 3 تلاش ناموفق
- قفل 5 دقیقه‌ای برای حساب‌های قفل شده

### 3. مدیریت پروژه‌ها (`project_manager.py`)

#### کلاس `ProjectManager`
مسئول مدیریت کامل پروژه‌ها و شرکت‌ها.

**متدهای اصلی:**
- `load_projects()`: بارگذاری پروژه‌ها از فایل
- `save_projects()`: ذخیره پروژه‌ها در فایل
- `show_project_management()`: نمایش پنجره مدیریت پروژه
- `create_projects_table()`: ایجاد جدول پروژه‌ها
- `show_add_project_form()`: نمایش فرم افزودن پروژه
- `show_add_company_form()`: نمایش فرم افزودن شرکت
- `delete_project()`: حذف پروژه
- `show_project_details()`: نمایش جزئیات پروژه

**اطلاعات پروژه:**
- نام پروژه، کارفرما، تاریخ شروع و پایان
- درآمد، هزینه و سود خالص
- اعضای تیم و شرح پروژه
- وضعیت پروژه (در حال اجرا، پایان یافته)

### 4. مدیریت گزارش‌ها (`reports_manager.py`)

#### کلاس `ReportsManager`
مسئول تولید انواع مختلف گزارش‌ها از پروژه‌ها.

**انواع گزارش‌ها:**
- گزارش بر اساس نام پروژه
- گزارش بر اساس نام شرکت/کارفرما
- گزارش بر اساس تاریخ شروع
- گزارش بر اساس تاریخ پایان
- گزارش پروژه‌های یک ماه خاص
- گزارش مالی هفتگی
- گزارش مالی ماهانه
- گزارش مالی سالانه

**متدهای اصلی:**
- `show_reports_window()`: نمایش پنجره گزارش‌گیری
- `clear_results()`: پاک کردن نتایج قبلی
- `add_project_to_results()`: افزودن پروژه به نتایج
- `update_count_label()`: به‌روزرسانی تعداد نتایج

### 5. ویرایشگر متن (`text_editor.py`)

#### کلاس `TextEditor`
ویرایشگر متن پیشرفته با قابلیت‌های متنوع.

**ویژگی‌ها:**
- برش، کپی و چسباندن متن
- تغییر رنگ متن و پس‌زمینه
- تغییر فونت و اندازه
- عملیات بازگشت و تکرار
- کلیدهای میانبر (Ctrl+S, Ctrl+Z, Ctrl+Y)

**متدهای اصلی:**
- `show_editor()`: نمایش ویرایشگر
- `create_toolbar()`: ایجاد نوار ابزار
- `save_text()`: ذخیره متن
- `get_text()`: دریافت متن فعلی

## 🚀 نحوه اجرا

- Python 3.6 یا بالاتر
- کتابخانه‌های استاندارد Python (tkinter, json, os, datetime, hashlib)

### نصب و اجرا
1. **دانلود پروژه:**
   ```bash
   git clone [آدرس مخزن]
   cd طراحی-و-پیادهسازی-نرمافزار-مدیریت-پروژههای-نرمافزاری
   ```

2. **اجرای برنامه:**
   ```bash
   python main.py
   ```

3. **یا با استفاده از Python3:**
   ```bash
   python3 main.py
   ```

## 📱 راهنمای استفاده

### 1. ورود به سیستم
- روی دکمه "ورود به سیستم" کلیک کنید
- نام کاربری و رمز عبور خود را وارد کنید
- در صورت فراموشی رمز، از دکمه "ثبت‌نام" استفاده کنید

### 2. ثبت‌نام کاربر جدید
- روی دکمه "ثبت‌نام" کلیک کنید
- نام کاربری (حداقل 3 کاراکتر) و رمز عبور (حداقل 6 کاراکتر) وارد کنید
- رمز عبور را تکرار کنید

### 3. مدیریت پروژه‌ها
- پس از ورود، روی "مدیریت پروژه" کلیک کنید
- برای افزودن پروژه جدید: "ثبت پروژه جدید"
- برای ثبت شرکت: "ثبت اطلاعات شرکت/کارفرما"
- برای ویرایش: پروژه را انتخاب کرده و "ویرایش پروژه" را کلیک کنید
- برای حذف: پروژه را انتخاب کرده و "حذف پروژه" را کلیک کنید

### 4. گزارش‌گیری
- روی "گزارش‌گیری" کلیک کنید
- نوع گزارش مورد نظر را انتخاب کنید
- پارامترهای جستجو را وارد کنید
- نتایج در جدول نمایش داده می‌شود

### 5. ویرایشگر متن
- در فرم‌های مختلف، از دکمه‌های ویرایش متن استفاده کنید
- امکان تغییر رنگ، فونت و اندازه متن
- کلیدهای میانبر برای عملیات سریع

## 🔒 ویژگی‌های امنیتی

- **رمزنگاری رمز عبور**: استفاده از الگوریتم SHA-256
- **قفل حساب**: پس از 3 تلاش ناموفق ورود
- **مدت قفل**: 5 دقیقه برای حساب‌های قفل شده
- **ذخیره امن**: اطلاعات در فایل‌های JSON محلی

## 🎨 رابط کاربری

- **طراحی فارسی**: مناسب برای کاربران فارسی‌زبان
- **رنگ‌بندی زیبا**: استفاده از پالت رنگی مدرن
- **انیمیشن**: افکت‌های hover برای دکمه‌ها
- **پنجره‌های مدال**: جلوگیری از تداخل با پنجره اصلی

## 🛠️ توسعه و سفارشی‌سازی

### افزودن ویژگی جدید
1. کلاس جدید را در فایل جداگانه ایجاد کنید
2. آن را در `main.py` import کنید
3. دکمه مربوطه را به منو اضافه کنید
4. متد مربوطه را در کلاس اصلی پیاده‌سازی کنید

### تغییر ظاهر
- فونت‌ها در متد `setup_fonts()` قابل تغییر هستند
- رنگ‌ها در متغیرهای مختلف تعریف شده‌اند
- اندازه پنجره‌ها در `geometry()` قابل تنظیم است
